# KaxxBoard
Advanced Scoreboard Management API for your Minecraft Bukkit plugins.

*Tested Minecraft versions: 1.8.8*  
**You can use this as a library** (see [the docs](https://kaxx.gitbook.io/kaxxboard/))

## Features
* Easy to use
* Customizable
* Packet based
* Asynchronous
